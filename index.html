数当てゲームとは、１〜9のランダムな数を予想して3回以内に的中させるゲームです。 予想を外すとコンピュータが正解の数が予想の数字よりも上か下かをおしえてくれます 3回予想を外してしまったらそこで答えを教えてくれてゲーム終了となります このゲームはプログラミング初心者にとってjavaScriptなどの基礎的な部分を扱うのでとても理解しやすいプロジェクトになっていると思います このプロジェクトを通じて、JavaScriptの基礎をしっかりと学び、それを実際のアプリケーションに応用する力を身につけることができました。 また、HTMLとの組み合わせによって、ウェブ上での実装の幅を広げることができたと思います このゲームを作るのにとても苦労したため完成して動いた時はとても感動しました

各授業の学習事項

第１回WWWとHTML

Webページを提供する仕組みであるWWWとそれを構成する様々な技術要素、見出しや本文図表などの構図を記述するHTML文書の作成 今回の講義を受講することによってWebページに掲載するコンテンツと、ページの基本的なレイアウトを設定できるようになります

演習の成果： 演習１−４：HTMLの様々な要素

第2回CSSによるWebページの装飾

前回の講義で学んだHTMLの要素ごとにそれぞれの見た目（装飾）を設定できるようになりWebページにおける色の区別や見やすいWebページを作成するための基本的な技術要素を学びました

演習の成果： 演習２−３:ウェブブラウザぺ説明ページの装飾

第3回JavaScriptの基本

JavaScriptというWebページ上で複雑な機能を実現するプログラミング言語を学びました この講義を受講することでページの自動更新やアニメーションなど動きのあるページを実現するためのプログラミングが可能になります．

演習の成果： 演習3-6:オブジェクトの参照

第4回DOMとイベント処理

DOM（Document Object Model）は Web コンテンツを表現するデータ構造です。DOM を使用することによって、JavaScript などのプログラムによって、Web コンテンツの内容を読み取ったり，コンテンツに変更を加えたりする処理が書けるようになります． 前回の講義で学んだ JavaScriptと併用することで動きのあるWebページが作成できます

演習の成果： 演習４−２:tashizan.jsの作成

第5回dom(2)

今回の講義を受講することで前回でもやったDomへの理解を深めることができました Domをマスターすることによって、JavaScriptを使って自由にHTMLのページを操作することができるようになります 入力に応じて内容を更新したり削除する処理を学び、より実践的な内容を学べてと思います

演習の成果：演習5-7:DOMのプログラミング

感想

Webページにはどのような要素があって、どのようなプログラムで動作しているかを学習しました。一つ一つの要素の役割についてや、どのような機能を持っているのかを講義を通して学び、Webページに対する理解を深めることができました。特にHTML・CSS・JavaScriptというフロントエンドの基本的な三つの技術について、それぞれの特徴や役割を知ることで、今まで意識せずに見ていたWebページがどのように構成され、動いているのかが少しずつ見えるようになりました。 1回目の講義ではHTMLの基本的な要素を学び、ページの構造をどのように記述するのかを知ることができました。2回目の講義ではCSSを使ったデザインの調整方法を学び、Webページの見た目を整えるための知識を身につけました。3回目以降ではJavaScriptを用いて、ボタンを押すと反応するプログラムなど、動きのあるページの作り方を学び、最終的にはDOM操作など応用的な内容にも触れることができました。これらはすべて今後の学習や将来のキャリアにも大きくつながる内容だと実感しています。 最初は「Webページを作る」と言っても、何から始めればいいのか分からず、不安もありましたが、講義を重ねていく中で少しずつ構造を理解し、その奥深さや、ひとつひとつの要素がいかに重要かを実感しました。実際に手を動かしていく中で、「Webってこうやってできてるんだ」と気づけたときの感動は、今でも印象に残っています。 もちろん、まだまだ知らないことや理解が曖昧な部分も多く、作れるものには限界がありますが、基本がしっかり身についていけば、より自由で実用的なWebサイトを作ることも可能になると思います。今後はより複雑なWebアプリにも挑戦したいと考えていますし、自分のアイデアを形にできるようになりたいとも思っています。 今回のメディア実験を通して、それまで何も考えずに使っていたWebページに対して「これはどうやって動いているのだろう」と考えるようになり、より深い視点からWebを見るようになりました。途中、何度もつまずき、思うように動かない場面も多くありましたが、そのたびに講義資料を読み返したり、エラーの原因を調べたりすることで少しずつ成長を感じました。 今後はさらにHTML、CSS、JavaScriptについての理解を深め、応用的な内容にも積極的に取り組んでいきたいです。そしていずれは、誰かの役に立つような、実用性のあるWebサービスを作ることができるようになりたいと思っています。今回の学びはそのための確かな第一歩となりました。
